# Riyadh Real Estate Data Analysis

## 📌 Overview

This project analyzes real estate data across Riyadh neighborhoods to understand pricing patterns and the factors affecting property values.

The analysis was performed using SQL Server, and results were visualized through an interactive Power BI dashboard.

---
## 🎯 Objectives

* Identify the most and least expensive regions and neighborhoods
* Analyze factors affecting property prices
* Explore price distribution across Riyadh
* Support decision-making in real estate investments

---
## 📊 Data

* Neighborhoods across Riyadh
* Property features (area, rooms, bathrooms, floors, furnishing, elevator)
* Property age
* Price and price per square meter

---
## 🧠 Analysis Performed

* Aggregations (AVG, COUNT, MIN, MAX)
* Ranking (Top / Bottom properties)
* Correlation analysis (price vs features)
* Regional comparisons

---
## 📈 Key Insights

* North Riyadh has the highest property prices, while the South has the lowest
* Al-Nakheel Al-Sharqi is the most expensive neighborhood
* Al-Mansoura is the least expensive and has the highest number of properties
* Larger properties (area) strongly correlate with higher prices
* Property age negatively impacts price (older = cheaper)
* Features like elevators and number of rooms increase property value
* Price per square meter is a strong indicator but not the only factor

---
## 📊 Dashboard

An interactive Power BI dashboard was created to visualize:

* Price distribution by region and neighborhood
* Top and lowest properties
* Price per square meter
* Impact of property features on price
